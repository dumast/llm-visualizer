# Frontend
## When running for the first time:
- cd frontend
- yarn install

## Normally:
- cd frontend
- yarn run dev

# Backend
## When running for the first time:
- cd backend
- python3 -m venv venv
- source venv/bin/activate (linux/macos) | venv\Scripts\activate (windows)
- pip install -r requirements.txt

## Normally:
- cd backend
- source venv/bin/activate (linux/macos) | venv\Scripts\activate (windows)
- python run.py
